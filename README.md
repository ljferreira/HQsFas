# HQsFas (Heróis dos Quadrinhos - Fãs)
Acessível em https://h-qs-fas.vercel.app/  
Landing Page que exibe informações sobre personagens dos quadrinhos.  
Possibilita a pesquisa de personagens pelas principais produtoras ( Marvel, DC, Dark Horse, NBC Heroes ), ou por Todas existentes pela API akabab/superhero-api. Também é possível localizar o personagem pelo nome ou suas iniciais.
